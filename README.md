# Playtime

PlayTime plugin is a simple plugin for PocketMine that allows you to check the amount of time a player has spent on the server.

### Commands:

* /playtime
* /playtime (player)

### API:
```php
PlayTime::getInstance()->getPlayTime($player);
```
